# Live2D AI 助手

这是一个基于Live2D和AI技术的虚拟助手项目，可以与用户进行对话交互并通过Live2D模型展示。

## 功能特点

- 集成Live2D模型展示
- 基于Ollama的AI对话能力
- 文本转语音(TTS)功能
- 可自定义背景和模型
- 多种Live2D模型可选

## 使用方法

1. 打开live2d_llm.html文件
2. 选择喜欢的Live2D模型
3. 调整AI设置和角色设定
4. 开始与AI助手对话

## 技术栈

- Live2D Cubism SDK
- PIXI.js
- jQuery
- Web API
